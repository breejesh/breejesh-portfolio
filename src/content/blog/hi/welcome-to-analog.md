# Analog.js में आपका स्वागत है!

नमस्ते! मैंने अपनी पोर्टफोलियो वेबसाइट को मानक Angular CLI से **Analog.js** और **Vite** पर सफलतापूर्वक माइग्रेट कर लिया है। यह मुझे तेज़ बंडलिंग और फ़ाइल-सिस्टम आधारित राउटर की सुविधा प्रदान करता है।

यहाँ सिंटैक्स हाइलाइटिंग की जाँच के लिए एक उदाहरण दिया गया है:

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  standalone: true,
  template: `<h1>नमस्ते Analog!</h1>`
})
export class AppComponent {
  title = 'Analog.js पोर्टफोलियो';
}
```
