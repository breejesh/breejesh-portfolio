### Qué es realmente Grok 4.5

SpaceXAI lanzó **Grok 4.5** el **8 de julio de 2026**. El acceso de consumo va en SuperGrok: apps/web, límites altos, razonamiento, visión, Imagine y coding con Grok Build. Write-ups de terceros citan una escala de **~1.5T parámetros** (divulgación del lab, no pesos abiertos).

Precio de API (aparte de SuperGrok), según [Artificial Analysis](https://artificialanalysis.ai/models/grok-4-5):

- **$2 / 1M input**, **$6 / 1M output**
- Cache hit ~**$0.50 / 1M** (-75%)
- Contexto: **500k tokens**
- Multimodal: texto + imagen in, texto out

2. **Elo de coding pico.** Fable 5 sigue delante en Coding Agent Index.
3. **Alucinaciones.** AA notó más accuracy *y* más alucinación vs Grok previo.
4. **Contexto.** 500k es grande; algunos anuncian 1M.
5. **Los precios se mueven.** Revisa el checkout.
