# ¡Bienvenido a Analog.js!

¡Hola! He migrado con éxito mi sitio web de portafolio desde la interfaz de línea de comandos estándar de Angular a **Analog.js** y **Vite**. Esto me proporciona la velocidad de empaquetado de Vite y enrutamiento basado en archivos.

Aquí hay un bloque de código de ejemplo para verificar la sintaxis:

```typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  standalone: true,
  template: `<h1>¡Hola Analog!</h1>`
})
export class AppComponent {
  title = 'Portafolio Analog.js';
}
```
